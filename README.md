# sneaky snake
snake game using python (Many rules differ from regular Snake you know)

Objects in games:
- Snake (Main Character)
- Rabbit (Grow the snake)
- Eden Apple (Offer boost) (good or evil?)
- Hunter (Kill the snake)

Purpose:
Try to survive one year
